# Python Data Analysis Projects

## Overview
This repository is a collection of practical data analysis projects and academic lab assignments completed during my Software Engineering studies. It serves as a portfolio demonstrating my progression in applying Python for data manipulation, statistical analysis, and visualization.

## Key Skills & Methods Demonstrated
Throughout these notebooks, I applied various techniques to extract insights from raw data:

* Handling missing values, correcting data types, standardization, and outlier detection/filtering using **Pandas**.
* Investigating data structures, descriptive statistics, and identifying key patterns.
* Building insightful charts (boxplots, histograms, scatter plots) to communicate findings using **Matplotlib**.
* Applying mathematical statistics, including hypothesis testing, ANOVA, and correlation analysis using **NumPy**.
* Creating new calculated metrics (e.g., population density) to enrich datasets for deeper analysis.

## Technologies Used
* **Language:** Python
* **Libraries:** Pandas, NumPy, Matplotlib
* **Environment:** Jupyter Notebook
